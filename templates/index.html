About TrustScan

TrustScan is an AI-powered web application designed to help users identify fake reviews before purchasing products online.

🚀 Built using: - Flask (Backend) - SerpAPI (Data fetching) - HTML, CSS (Frontend)

🎯 Goal: To improve online shopping decisions by providing transparency.

💡 Future Scope: - Real ML model for accuracy - Chrome extension - Mobile app version
